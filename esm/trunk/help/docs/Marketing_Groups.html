Introduction to Marketing Groups

Marketing groups are used to funnel the 20 or so Imis groups into more manageable entities such that managing site content is easier.

Important Concept : Marketing groups are made of individual Imis groups. An Imis group can only belong to one marketing group.

 

Adding a Marketing Group

To add a Marketing Group, follow these directions:

  1. Click on the Marketing Groups tab in the main navigation.
  2. Click on the Add Marketing Group link just below the main navigation.
  3. Fill in the form. See Field Definitions for more info.
  4. Click Save to commit your entry.

Editing a Marketing Group

To edit a Marketing Group, follow these directions:

  1. Click on the Marketing Groups tab in the main navigation.
  2. Find the marketing group that you wish to edit by either clicking on it in the left navigation or using the search in the left column.
  3. Update the form. See Field Definitions for more info.
  4. Click Save to commit your updates.

 

Deleting a Marketing Group

To delete a Marketing Group, follow these directions:

  1. Click on the Marketing Groups tab in the main navigation.
  2. Find the marketing group that you wish to delete by either clicking on it in the left navigation or using the search in the left column.
  3. Click the Delete button.

Note : A marketing group may not be deleted if imis groups are associated with it or if it is embedded in a page. Remove the association to delete the group.

 

Adding an Imis Group

To add an Imis Group, follow these directions:

  1. Click on the Marketing Groups tab in the main navigation.
  2. Click on the Add Member Type link just below the main navigation.
  3. Fill in the form. See Field Definitions for more info.
  4. Click Save to commit your entry.

 

Editing an Imis Group

To edit an Imis Group, follow these directions:

  1. Click on the Marketing Groups tab in the main navigation.
  2. Click on the Member Types link just below the main navigation.
  3. Find the imis group that you wish to edit in the left navigation.
  4. Update the form. See Field Definitions for more info.
  5. Click Save to commit your updates.

Deleting an Imis Group

To delete an Imis Group, follow these directions:

  1. Click on the Marketing Groups tab in the main navigation.
  2. Click on the Member Types link just below the main navigation.
  3. Find the Imis group that you wish to delete by either clicking on it in the left navigation or using the search in the left column.
  4. Click the Delete button.

Note : A marketing group may not be deleted if imis groups are associated with it or if it is embedded in a page. Remove the association to delete the group.

 

Field Definitions

Marketing Groups

Marketing Group Name : The name for this marketing group. This should be meaningful such that it can be easily recognized when adding content.

Description : A short description for this group. Only visible while editing marketing groups.

Member Types : All of the Imis groups associated with this group.

Imis Groups

Imis Group Name : The abbreviation for this Imis group. This should be written exactly as it will come across via the ISG webservices.

Description : A short description for this group. Only visible while editing this Imis group.
